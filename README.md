# 八字AI分析工具

一个基于纯前端的八字计算和AI分析工具，完全免费使用。

## 功能特点

1. 🎯 **八字计算**：根据出生时间计算四柱八字
2. 🤖 **AI分析**：智能分析八字，解答各类问题
3. 📱 **响应式设计**：支持手机、平板、电脑
4. 🆓 **完全免费**：无需付费，无隐藏费用
5. 🔒 **隐私保护**：所有计算在浏览器完成，数据不上传

## 使用方法

1. 输入公历生日
2. 选择出生时辰
3. 选择性别
4. 点击"计算八字"
5. 在下方输入问题，点击"AI智能分析"

## 技术栈

- HTML5
- CSS3
- JavaScript
- 纯前端计算
