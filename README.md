# 八字分析平台

这是一个基于传统八字理论与现代人生规划相结合的免费分析工具。

## 功能特点

- 支持公历/农历输入
- 使用lunar-javascript库进行精确的八字计算
- 基于专业规则库进行分析
- 将传统八字术语转化为现代人生规划语言
- 完全前端实现，无需服务器支持

## 使用说明

1. 输入您的出生年月日时信息
2. 选择是否为农历日期
3. 点击"开始分析"按钮
4. 查看详细的分析结果

## 文件结构

- index.html - 主页面
- style.css - 样式文件
- script.js - 主要功能代码
- lunar.js - 八字计算库（需要从lunar-javascript库获取）

## 注意事项

- 本工具仅供娱乐参考，不构成专业建议
- 计算结果基于传统八字理论，不保证100%准确
- 请勿用于商业决策、医疗诊断等重要事项

## 技术实现

- 使用纯前端技术，无需后端支持
- 基于lunar-javascript库进行八字计算
- 响应式设计，支持各种设备访问
