# 八字潜能分析平台（纯前端版）

## 📌 项目简介
本项目是一个完全基于浏览器的免费八字分析工具，用户输入出生日期后：
1. 本地计算四柱八字（年、月、日、时干支）
2. 结合传统命理规则库（`rules.json`）生成个性化潜能解读
3. 输出符合合规要求的现代人生规划建议

## ⚠️ 重要说明
- **不依赖任何后端服务**，所有计算在用户浏览器完成
- **八字排盘基于公开算法**，精度满足一般参考需求
- **分析逻辑源自上传的参数库**，已做合规化转译
- **严禁用于医疗、法律、投资等重大决策**

## 🛠 技术栈
- HTML5 + CSS3 + JavaScript (ES6)
- 无第三方框架（零依赖）
- 本地存储规则库（`rules.json`）
